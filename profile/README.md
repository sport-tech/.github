# Sport‑Tech — Organization profile

Welcome to Sport‑Tech — a developer and maker hub for sport-technology products that help athletes train smarter and push performance boundaries.

Current focus — Fatalita (table-tennis training robot)
- Working name: Fatalita
- Short description: A configurable, programmable ball-delivery robot for table-tennis training. Designed to help players practice strokes, placement, footwork and rally patterns.
- Key capabilities (planned/ongoing):
  - Configurable spin, speed and placement
  - Programmable drill sequences and randomization
  - Networked UI and telemetry for drill creation and performance tracking
  - Expandable modules for different ball-types and sensors
- Goal: iterate quickly, publish SDKs and reproducible hardware/firmware designs, then expand to other sport-tech products.
