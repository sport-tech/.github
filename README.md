# .github (Sport‑Tech central metadata)

This repository hosts organization-wide meta files and templates for Sport‑Tech. It provides common configuration, templates and guidance used across our projects.

What this repository contains (recommended)
- profile/README.md — Organization profile shown on GitHub (this file).
- ISSUE_TEMPLATE/ and PULL_REQUEST_TEMPLATE/ — templates to standardize issues and PRs.
- CONTRIBUTING.md — global guidance for contributors (can be supplemented per-repo).
- CODE_OF_CONDUCT.md — community standards.
- workflows/ — shared GitHub Actions workflows (optional).
- funding.yml / sponsors — sponsorship and funding information.
